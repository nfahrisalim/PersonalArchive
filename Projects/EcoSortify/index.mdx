EcoSortify is an interactive and eco-friendly educational waste sorting application designed to increase public awareness in waste management through innovative digital approaches. This application utilizes AI technology based on MobileNetV2 to automatically identify waste types (organic, inorganic, and hazardous B3) through smartphone cameras.

The application is equipped with outstanding features such as waste classification using machine learning, EcoBot as an AI assistant for sorting guidance, educational mini-games to enhance understanding, and a plogging tracker feature that allows users to track jogging activities while picking up trash with the help of GPS and AI detection.
