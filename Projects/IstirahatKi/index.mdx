## Overview

IstirahatKi is an Android app that helps build healthy habits for hydration and adequate sleep. Born from a developer's personal struggle with forgetting to drink water and sleeping late due to work demands, this app combines dual tracking system with gamification to boost user motivation. The clean, user-friendly interface encourages users to maintain life balance through gamified habit tracking.

## Technical Implementation

Built with **MVVM architecture** using Android SDK 35 and Java 8, implementing Material Design 3 for consistent theming across light/dark modes. The app uses SQLite with custom DatabaseHelper for local storage, Retrofit + Gson for API communications, WilliamChart for data visualization, and Lottie for smooth micro-interactions. ViewBinding ensures type-safe view references while LiveData & Observer pattern enables reactive programming. Background services handle smart notification system with haptic feedback for enhanced user experience.

## Target Audience
